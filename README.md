# Plataforma FARSEB

Web informativa de código abierto que recopila y organiza normativa laboral española basada en fuentes oficiales del Boletín Oficial del Estado (BOE), el Ministerio de Trabajo y Economía Social, la Seguridad Social y la Agencia Tributaria.

## Contenido
- Resumen de la legislación laboral fundamental.
- Derechos laborales clave para las personas trabajadoras.
- Obligaciones empresariales en materia laboral y fiscal.
- Procedimientos esenciales publicados en el BOE.
- Enlaces a recursos oficiales actualizados.

## Tecnologías
- HTML5 semántico.
- CSS3 responsivo con estilos modernos.
- JavaScript ligero para interacciones accesibles.

## Uso
Abrir `index.html` en el navegador para explorar la información.
